# Security Policy / 安全策略

请勿在公开 Issue 中提交账号、Cookie、Token、二维码会话、抓包文件或签名密钥。

安全问题请发送至 **xutianyoubupt@foxmail.com**，邮件包含受影响版本、复现步骤、影响范围和建议修复方式。收到报告后会先确认影响，再协调修复与披露时间。

QuizPane 的 Provider 以原生动态库形式运行，安装来源不明的 Provider 等同于运行第三方本地代码。正式发行版默认只应信任经过审核和签名的题库组件。

Do not disclose credentials, cookies, tokens, QR sessions, packet captures, or signing keys in public issues. Report vulnerabilities privately to **xutianyoubupt@foxmail.com** with the affected version, reproduction steps, impact, and any suggested remediation.
